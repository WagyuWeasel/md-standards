## A

### API key

A unique string that authenticates requests to the Fastlog API. API
keys are scoped to a single project and can have read-only or
read-write permissions. Generate keys in the Dashboard under Settings >
API Keys.

See also: [Authentication](#authentication)

### Authentication

The process of verifying the identity of an API caller. Fastlog
supports two methods: API keys for server-to-server calls and OAuth
2.0 tokens for user-facing applications.

## B

### Batch ingestion

A method of sending multiple log entries in a single API request.
Batch ingestion reduces network overhead and is recommended for
high-throughput applications. The maximum batch size is 1,000 entries.

See also: [Streaming Ingestion](#streaming-ingestion)
